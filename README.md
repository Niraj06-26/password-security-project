# Password Security Project

## Overview

This project demonstrates a Dictionary Attack to understand password security weaknesses.

## Objective

- Understand Password Attacks
- Demonstrate Dictionary Attack
- Identify Weak Passwords
- Improve Security Awareness

## Tools Used

- Kali Linux
- Hydra
- RockYou Wordlist

## Methodology

1. Select Target Service
2. Load Password Wordlist
3. Run Dictionary Attack
4. Analyze Results

## Results

- Weak Passwords Identified
- Dictionary Attack Demonstrated
- Security Risks Analyzed

## Security Recommendations

- Use Strong Passwords
- Enable MFA
- Avoid Common Passwords
- Use Password Managers

## Skills Learned

- Password Security
- Authentication Security
- Ethical Hacking Concepts
